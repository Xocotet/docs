# Analytics

mmesh monitors the nodes out-of-the-box. No configuration is required.

It monitors the following node metrics:

- Load Average (not supported by Windows)
- CPU Usage
- Memory Usage
- Disk Usage (main filesystem)
- Network Usage (mmesh traffic)
